# Online DND Sites

* Start Playing Games
  * StartPlaying is the largest online platform for players to find tabletop roleplaying games and professional GMs for any game system and any virtual tabletop!
  * Includes support for getting payed to DM
  * https://startplaying.games/
* Home Crit
  * https://www.naturalcrit.com/
    
## VTT Resources

* Roll20
  * Online VTT
  * Good integration with payed resources
  * https://roll20.net/
* Foundry VTT
  * Slick interface
  * Self hosted
  * VTT marketplace - Limited content relative to Roll20 and FG?
  * https://foundryvtt.com/
* Forge
  * Online Foundry VTT server
  * Players free
  * https://forge-vtt.com/
  * Roll20 converter
    * Convert content from Roll20
    * https://github.com/kakaroto/R20Converter
* TablePlop
  *   Online VTT
  * Very simple, which seems a plus
  * No apparent integration with paid resources
  * Free
  * Small user community
  * Paetron integration
  * https://new.tableplop.com/
    
## Character Manager

* Demiplane
  * A character sheet application that supports many RPGs
  * Includes DND Nexus, to bring in DND content (e.g. books)
  * Includes Roll20 Integration
  * Subscription
  * https://app.demiplane.com/home
